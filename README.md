# Simulaci-n-de-sistema-de-control-de-lazo-cerrado-para-sistema-de-refrigeraci-n-de-PC-domestica-

Programa que emula el sistema de control termico de la cpu de una PC. Muestra graficos para visualizar los valores de temperatura real, temperatura objetivo, error y porcentaje de CPU utilizado. Permite modificar ciertas variables como la potencia de las perturbaciones, la efectividad de los controles y el valor de temperatura esperado.

Para utilizarlo solo hay que descargar el .exe y ejecutarlo.
